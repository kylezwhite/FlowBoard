Connect to Google

FlowBoard needs a Google OAuth Client ID to securely connect to Drive. You only do this once — your key is saved in your browser.

Get your free Client ID (10 min):
1. Visit console.cloud.google.com → create a project
2. Enable Google Drive API under APIs & Services
3. Go to Credentials → Create → OAuth 2.0 Client ID
4. Pick Web application → add your site URL as authorized origin
5. Copy the Client ID and paste it below
