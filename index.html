Connect to Google

Taskie needs a Google OAuth Client ID. This is free and takes about 10 minutes to set up once.

How to get your Client ID:
1. Go to console.cloud.google.com → create a project
2. Enable Google Drive API under APIs & Services
3. Go to Google Auth Platform → Get started (choose External)
4. Go to Clients → Create Client → Web application
5. Add your site URL under Authorized JavaScript Origins
6. Copy your Client ID and paste below

Transfer Org Ownership

How this works:

Taskie stores org data in the Google Drive of whoever originally created the org. Transferring ownership has two parts:

Step 1 — Role transfer (automatic): The new owner gets the Owner role. You become an Admin and stay in the org.

Step 2 — Drive data transfer (manual, recommended):
We'll grant the new owner full access to all org files. To fully move the data to their Drive:
1. Ask the new owner to sign into Google Drive and find the Taskie_Orgs folder shared with them
2. They right-click it → Make a copy (or Add shortcut to Drive)
3. Verify Taskie still works correctly for all members
4. You can then remove the original folder from your Drive if desired

⚠ Do not delete the original until the new owner confirms everything is working.
